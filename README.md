## 🧩 What anolisa is

ANOLISA stands for Agentic Nexus Operating Layer & Interface System Architecture. It is an end-user app that helps you run a local agentic workspace on Windows. It gives you one place to launch tasks, manage skills, and keep your work organized.

Use it when you want a simple interface for agent-based tools without setting up a full technical stack.

## 💻 What you need

anolisa works on Windows 10 and Windows 11.

You should have:

- A Windows PC with admin access
- At least 8 GB of RAM
- 2 GB of free disk space
- A stable internet connection for the first download
- A mouse and keyboard

For best use, keep Windows up to date.
